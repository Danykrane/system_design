## Почему WebRTC + SFU

| Требование | Почему важно | Выбор |
|---|---|---|
| Низкая задержка аудио/видео | Основной UX фактор для звонков | WebRTC media plane |
| Масштабирование групповых комнат | P2P плохо масштабируется по uplink | SFU вместо mesh |
| Адаптация качества под сеть | Разные каналы/устройства | Simulcast/SVC + GCC |
| Кроссплатформенность | Web + Desktop + Mobile | Единый WebRTC стек |
